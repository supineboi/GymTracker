## Learning Path Overview

We'll build a GymTracker app step-by-step, learning React concepts as we go!

### **Lesson 1: Component Structure & JSX Basics**
- Clean up the starter code
- Create our first custom component
- Learn JSX syntax and why we use it
- Understand component composition

### **Lesson 2: State Management with useState**
- Track workout data using state
- Understand why state is needed
- Learn how React re-renders components

### **Lesson 3: Props - Passing Data Between Components**
- Create reusable components
- Pass data from parent to child
- Understand one-way data flow

### **Lesson 4: Event Handling**
- Handle user interactions (add, delete workouts)
- Understand synthetic events
- Learn event handlers in React

### **Lesson 5: Lists & Keys**
- Render lists of workouts/exercises
- Understand why keys are important
- Learn map() for rendering arrays

### **Lesson 6: Conditional Rendering**
- Show/hide UI based on state
- Use ternary operators and && operator
- Create better UX with conditional displays

### **Lesson 7: Forms & Controlled Inputs**
- Create forms for adding workouts
- Understand controlled vs uncontrolled components
- Handle form submission

### **Lesson 8: Data Persistence with localStorage**
- Save data to browser storage
- Load data on app start
- Understand side effects
